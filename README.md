# resource-sites-bet365

本仓库用于归档和发布多个独立 HTML 页面，内容涵盖各类静态资源页面，不针对任何单个域名或具体网站。

## 项目简介

本项目旨在提供一个集中存放与展示独立 HTML 页面的仓库，方便用户浏览、索引或引用。所有页面均为纯前端静态实现，无需后端服务支持。

## 目录说明

```
/
├── index.html        # 项目入口页面（导航页）
├── pages/            # 存放各独立 HTML 页面
│   ├── page1.html
│   ├── page2.html
│   └── ...
├── assets/           # 公共资源文件（样式、脚本、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md         # 本文件
```

- `pages/`: 每个独立 HTML 页面存放于此，按需命名。
- `assets/`: 公共资源文件，供各页面引用。

## 页面归档说明

- 所有页面均为独立、自包含的 HTML 文件，可直接在浏览器中打开使用。
- 页面内容不涉及具体域名或网站推荐，仅提供通用功能或信息展示。
- 页面更新时，会保留历史版本或直接覆盖，具体视维护策略而定。

## 维护说明

- 欢迎提交 issue 或 pull request 以改进页面质量或修复问题。
- 新增页面时，请确保文件命名清晰，并更新 `index.html` 导航链接。
- 不建议在此仓库中存放敏感信息或商业推广内容。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接双击 `index.html` 或 `pages/` 下的任意 HTML 文件即可浏览。
3. 也可通过 GitHub Pages 或其他静态托管服务部署后在线访问。
